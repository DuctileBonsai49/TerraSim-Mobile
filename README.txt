TerraSim Demo
=============

Dies ist eine Demo-Version von TerraSim, einem interaktiven Sandbox-Spiel mit einer nachgestellten Welt (Planet Terra ohne Menschen).

Funktionen:
- Zerstörung per Werkzeug (Meteor, Explosion, Feuer)
- Platzierung von Bäumen und Häusern
- Punktezählung
- Sprache Deutsch/Englisch umschaltbar
- Speichern & Laden des Spielstands im eigenen .ahs-Dateiformat
  (verschleierte, aber lesbare Textdatei)

Bedienung:
- Werkzeug auswählen und auf die Karte klicken, um zu zerstören oder Objekte zu platzieren.
- Mit dem Button ".ahs exportieren" den Spielstand speichern.
- Mit dem Datei-Input die gespeicherte .ahs Datei laden.

Technik:
- HTML5, JavaScript (Canvas), CSS
- Lokale Speicherung nur über Export/Import
- Keine externen Abhängigkeiten

Starte das Spiel durch Öffnen von index.html im Browser.
